Linux Mastery Syllabus

Module 1: Linux Basics

  • Introduction to Linux distributions (Ubuntu, CentOS, Debian)
  • Understanding the Linux filesystem hierarchy (/bin, /etc, /var, /home)
  • Essential commands: ls, cd, pwd, cp, mv, rm, cat, less
  • File permissions and ownership: chmod, chown, umask
  • User and group management: adduser, passwd, id, groups

Module 2: Working with Files & Processes

  • File editing tools: nano, vim, grep, find, awk, sed
  • Disk usage and management: df, du, lsblk, mount, umount
  • Process management: ps, top, htop, kill, jobs, bg, fg
  • System monitoring: uptime, free, vmstat

Module 3: Networking in Linux

  • Network configuration: ip, ifconfig, ping, traceroute
  • Service testing: curl, wget, telnet, nc
  • Firewall basics: iptables, ufw
  • SSH and remote access: ssh, scp, rsync

Module 4: System Administration

  • Managing services: systemctl, service
  • Logs and troubleshooting: journalctl, dmesg, /var/log
  • Scheduling tasks: cron, at
  • Package management: apt, yum, dnf, zypper
  • User privileges: sudo, /etc/sudoers

Module 5: Shell Scripting & Automation

  • Writing basic shell scripts
  • Variables, loops, conditionals
  • Functions and script debugging
  • Automating tasks with scripts
  • Using bashrc and environment variables

Module 6: Security & Performance

  • SELinux/AppArmor basics
  • File integrity tools (md5sum, sha256sum)
  • Resource limits: ulimit, nice, renice
  • System tuning: sysctl
  • Auditing and intrusion detection (auditd)

Module 7: Advanced Linux Concepts

  • Kernel internals: namespaces, cgroups
  • Advanced networking: VLANs, bonding, nftables
  • Debugging tools: strace, lsof, perf, gdb
  • Building and compiling software from source
  • Linux in containers (how Docker uses Linux features)

Module 8: Capstone Projects

  • Write scripts to automate backups and monitoring
  • Configure a secure Linux server with firewall + SSH hardening
  • Build a custom Linux service with systemd
  • Tune a Linux system for high-performance workloads
  • Explore container runtime internals (Docker on Linux)

  • What is containerization? Difference between containers and VMs
  • Docker architecture: client, daemon, images, containers
  • Installing Docker on Linux/Windows/Mac
  • Running your first container (docker run hello-world)
  • Basic commands: docker ps, stop, rm, images, rmi

Module 2: Working with Images

  • Pulling images from Docker Hub (docker pull)
  • Building custom images with Dockerfile (docker build)
  • Understanding image layers and caching
  • Tagging and pushing images to registries
  • Best practices for writing Dockerfiles

Module 3: Container Lifecycle & Management

  • Running containers interactively (-it)
  • Executing commands inside containers (docker exec)
  • Viewing logs (docker logs)
  • Restart policies and container health checks
  • Cleaning up unused containers and images (prune)

Module 4: Volumes & Networking

  • Volumes for persistent storage
  • Bind mounts vs. named volumes
  • Networking basics: bridge, host, none
  • Custom networks (docker network create)
  • Linking containers and service discovery

Module 5: Docker Compose

  • Introduction to Compose and YAML syntax

  • Compose project examples (web + database stack)

Module 6: Advanced Docker

  • Multi-stage builds for optimized images

Module 7: Docker Security

  • Image scanning tools (Trivy, Clair)

  • Running containers as non-root users
  • Secrets management in Docker
  • Best practices for secure Dockerfiles

Module 8: Docker in CI/CD

  • Integrating Docker with Jenkins, GitHub Actions, GitLab CI
  • Building and pushing images automatically
  • Testing containers in pipelines
  • Versioning and tagging strategies
  • Deploying Dockerized apps to staging/production

Module 9: Docker Swarm & Orchestration

  • Introduction to Docker Swarm mode
  • Creating and managing services
  • Scaling and load balancing in Swarm
  • Overlay networks for multi-host communication
  • Comparing Swarm vs Kubernetes

Module 10: Capstone Projects

  • Build and run a Dockerized web app with database
  • Create a multi-container stack using Docker Compose
  • Secure an app with non-root containers and image scanning
  • Automate builds and deployments with CI/CD pipelines
  • Deploy a distributed app using Docker Swarm

Module 9: Cluster Lifecycle Management

  • High availability control plane setup
  • Cluster federation and multi-cloud deployments
  • Performance tuning for large-scale clusters

Module 10: Capstone Projects

  • Deploy a containerized web app with scaling and ingress
  • Secure workloads with RBAC and Network Policies
  • Set up monitoring with Prometheus + Grafana
  • Run a stateful database with Persistent Volumes
  • Implement a service mesh for traffic management
  • Provision and manage a production-grade cluster end-to-end

Module 1: Terraform Fundamentals

  • Introduction to Infrastructure as Code (IaC)
  • Terraform architecture: providers, resources, state

Module 2: Variables & Outputs

  • Outputs for sharing data between modules
  • Sensitive variables and secrets management

Module 3: State Management

  • Inspecting state (show, state list)

Module 4: Terraform Modules (Core Concepts)

Module 5: Advanced Module Design

  • Passing variables between modules
  • Using modules from Terraform Registry

Module 6: Multi-Cloud Infrastructure

Module 7: Terraform + Kubernetes Integration

Module 8: Policy & Security

  • Policy as Code with Sentinel and OPA

  • Auditing and logging Terraform actions

Module 9: CI/CD & Automation

  • Integrating Terraform with Jenkins, GitHub Actions, GitLab CI
  • Automated pipelines for plan and apply
  • Versioning and tagging infrastructure releases
  • Testing infrastructure with Terratest
  • Continuous delivery of infrastructure changes

Module 10: Capstone Projects

  • Build reusable modules for VPC, subnets, and EC2
  • Provision a Kubernetes cluster with Terraform modules
  • Secure infrastructure with RBAC and policy enforcement
  • Automate deployments with CI/CD pipelines
